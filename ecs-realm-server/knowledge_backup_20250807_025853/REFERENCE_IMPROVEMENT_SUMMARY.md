2. **실습 중심 학습**: 연습 문제와 해답으로 실제 구현 능력 향상
3. **문제 해결 능력**: 일반적인 실수와 해결방법으로 실전 대비

2. **실용성**: 실제 개발에서 마주치는 문제와 해결책 제시
3. **연결성**: 다음 학습 단계 명확히 제시
